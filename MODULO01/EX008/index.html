Outras Formatações

Codigo-fonte / Pré-formatação

Para códigos utilizados a tag <code>, utilizado para aplicar e dar significado de formatação de códigos.
Segue exemplo:

if num % 2 == 0: print(f"O numero {num} é par") else: print(f"O numero {num} é impar")

Porém podemos perceber que o código não ficou formado corretamente, para isso é necessário utilizar a tag <pre>

Segue agora o código formato corretamente:

        
    if num % 2 == 0:
        print(f"O numero {num} é par")
    else:
        print(f"O numero {num} é impar")
